Getting Started

Welcome to SecureSling!

SecureSling is a secure file sharing service that uses end-to-end encryption to protect your files. Here's what you need to know:

Upload Files

Encrypt and upload files with optional password protection

Download Files

Download files using secure links and decryption keys

Important: All encryption and decryption happens in your browser. The server never sees your unencrypted files.

Download Guide

Downloading Files

To download a file shared with you:

  1. Click on the secure download link provided by the sender
  2. If the file was encrypted, enter the password (ask the sender if you don't have it)
  3. Click "Download File" to decrypt and download
  4. For one-time downloads, the file will be deleted after your download
Note: If you lose the password, the file cannot be recovered. Always keep passwords secure.

Security & Privacy

How SecureSling Protects Your Files
End-to-End Encryption
  • AES-GCM 256-bit encryption
  • Encryption happens in your browser
  • Server only stores encrypted files
Privacy Features
  • No file content access by server
  • One-time download option
  • Automatic file expiration
Security Best Practices
  • Use strong, unique passwords for each file
  • Enable one-time download for sensitive files
  • Set reasonable expiration dates
  • Share download links securely (e.g., encrypted messaging)
  • Verify the recipient's identity before sharing sensitive files
